Mobile Computing Applications

The real power of mobile computing becomes apparent when mobile hardware, Software, and communications are optimally configured and used to accomplish a Specified mobile task

computing applications can generally be divided into two categories :

Horizontal

Horizontal applications have broad-based appeal and include software that performs functions such as:

  1. email
  2. Web browsing
  3. word processing
  4. scheduling
  5. contact management
  6. to-do lists
  7. messaging

Vertical

Vertical applications are industry-specific and only have appeal within the specific Industry for which the application was written. Vertical applications are commonly used in industries such as:

  1. retailing
  2. utilities
  3. warehousing
  4. shipping
  5. Medical
  6. law enforcement
  7. public safety
